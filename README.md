Contact Saver Application (Python + MySQL)
 
This is a console-based Contact Saver Application developed using Python and MySQL.
 
## Features
- Add new contact
- View all contacts
- Search contact
- Update contact
- Delete contact
 
## Technologies Used
- Python
- MySQL
- mysql-connector-python
 
## Files in this Project
- contact_saver.py → Python application
- database_queries.sql → MySQL queries to create database and table
